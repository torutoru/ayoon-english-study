1. 난이도 별 대화 관리
-> 현재, 난이도 별 대화 관리가 안된다
-> 난이도별 대화 관리 필요
-> 쉬움 선택하면 쉬움 대화 목록만 노출. 어려움 선택하면 어려움 대화 내용만 노출

2. 음성 나오는 중 페이지 이탈 시(react unmount), 스피커 종료
-> 현재, 음성이 나오는 중간에 페이지 이탈을 해도 스피커에서 이전 대화 내용이 계속 나오는 상황
-> 음성 재생 중 다른 페이지 혹은 페이지 이탈시, React Page Unmount 시 이전에 재생되던 플레이어 종료

3. 어려움 난이도는 속도가 1배속이고, 한국어를 읽는 모드가 아닌데 한국어를 계속 읽는 문제.
-> 한국어 읽기는 쉬움 모드에서만 읽어주는 거다.
-> (쉬움) 영어 재생속도 0.5, 영어 내용이 번역된 한국어 읽어주기(한국어는 1배속)
-> (중간) 영어 재생속도 0.8, 영어 내용이 번역된 한국어 읽어주기(한국어는 1배속)
-> (어려움) 영어 재생속도 1. 한국어 읽어주지 않고, 영어만 읽어줌

4. (쉬움) 영어 문장이 너무 긴 상황
-> 영어를 완전 처음하는 유아 상대로 할 수 있는 영어 단어와 문장구조
-> 너무 긴 문장보단 짧은 문장 형식으로 질문.
-> 대답도 짧은 문장 혹은 단어만 대답할 수 잇는 형식으로 질문.
-> 영어를 완전 처음배우는 미국아이 1살 ~ 2살 수준으로 질문 구성

5. (중간) 짧은 문장을 대답으로 할 수 있는 질문 난이도로 구성
-> 어느정도의 짧은 문장의 대답을 할 수 있는 문장으로 구성
-> 이것 또한 질문의 길이가 너무 길지 않았으면 함
-> 영어를 조금은 사용할 수 있는 미국아이 3~5살 수준으로 질문 구성

6. (어려움) 어느정도 문장으로 대답할 수 있는 질문 난이도로 구성
-> 영어를 사용할 수 있는 초등학생 미국아이 5~10살 사이 수준으로 질문 구성
